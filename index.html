Defined useful features with PMs and Devs. Worked across a range of disciplines including wireframing to creating pixel-perfect mockups & interactions. Boosted conversions by optimizing the UI, and achieved a 166% growth in user engagement within 3 months.

Designed an e-commerce website from scratch named 'InsaneDeals'. Built an intuitive user experience with a mobile-first mindset, while still keeping the interaction patterns consistent for desktop. Collaborated with the PM and the dev team to ideate, design, and implement new features.
